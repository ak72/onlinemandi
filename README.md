# onlinemandi
React Native Typescript Android App

## Installation
To get a local copy up and running follow these simple example steps.
 Clone the repo
   ```sh
1.   git clone https://github.com/ak72/onlinemandi.git
   ```
  Install NPM packages
  ```sh
2.   npm install
   ```
